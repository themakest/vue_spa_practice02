## Vuexの特徴
- stateで状態を持つ。
- gettersでstateを返す。
- actions内のcommitにて、mutationsのメソッドを実行する。
- mutationsにてstateを操作する。

Vuexでのデータの流れは Actions → Mutations → State → Vue Components
